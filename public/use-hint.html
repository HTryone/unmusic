解锁音乐使用提示

纯浏览器端加密音乐解密工具 — 解密全在本地完成，不上传任何文件。

支持格式

.ncm .qmc0 / .qmc2 / .qmc3 / .qmc4 / .qmc6 / .qmc8 .mflac / .mgg .mflac2 / .mgg2 / .mflac4 / .mgg4 .musicex .kgm .kgma .kgg / .kgg.flac .vpr .tkm .bkc .mogg .ygg

① 大部分格式：拖入即用

打开网页，把加密音乐文件拖入页面即可，无需任何设置。适用：.ncm（网易云）、.kgm、.mflac/.mgg、.qmc* 等绝大多数格式。

② QQ 音乐 musicex：先填 Cookie + 代理

.musicex 的解密密钥不在文件里，需要你的 VIP 登录态才能拿到。按下面顺序操作：

1
登录 QQ 音乐 VIP — 浏览器打开 y.qq.com 并登录你的 QQ 音乐 VIP 账号。
2
拿到 Cookie — 二选一：
· 手动：按 F12 → 网络 → 随便点个请求 → 请求头里复制 Cookie 整行。
· 一键（Windows）：下载下面的脚本，双击 .bat 从客户端内存提取，复制输出的 cookie。
3
填写设置 — 点右上角齿轮进入设置页：把 Cookie 粘到「QQ 音乐 Cookie」框；若提示 104003 缺 uin，再把 QQ 号填到「QQ 音乐 uin（QQ 号）」框。
4
填代理地址 — 开发调试（npm run serve）时内置代理可留空，其余情况（含本地使用）都需填 Cloudflare Worker 地址（见下方「部署代理」）。
5
拖入解密 — 返回主页，把 .musicex 文件拖入页面即开始解密。

一键取 Cookie（Windows 客户端，可选）

需先打开 QQ 音乐客户端并登录 VIP、播过一首 VIP 歌。双击 scan_qq_cookie.bat（脚本要和同目录的 .ps1 放一起），复制输出的 cookie 粘进设置页即可。脚本只扫本机内存，不联网、不上传，与手动 F12 复制等效。

⬇ 下载 scan_qq_cookie.bat ⬇ 下载 scan_qq_cookie.ps1

部署代理

解密 musicex 需要一个代理转发（开发调试 npm run serve 除外）。下载脚本部署到 Cloudflare Workers，再把地址填到设置页即可。

⬇ 下载 qq-proxy.js

  1. 登录 Cloudflare Workers，新建一个 Worker。
  2. 把 qq-proxy.js 内容粘贴进去，保存并部署，得到形如 https://xxx.workers.dev 的地址。
  3. 把该地址填到设置页「QQ 音乐 API 代理地址」框（例如 https://unmusic-proxy.xianshenghu363.workers.dev）。

③ 酷狗 KGG v5：先导入密钥

KGG 的密钥存在酷狗客户端里，需导入后才能解密。

  1. 拿到密钥 — 以下任一来源：酷狗本地的 KGMusicV3.db、OpenConverter 的 kgg.keys，或直接导出 .key 文件（格式 keyId$ekey）。
  2. 导入密钥 — 设置页「酷狗 KGG v5 密钥」处，拖入 .db / .key 文件。
  3. 拖入解密 — 首页拖入 .kgg 文件即自动解密。
⚠ 密钥与本账号绑定，需同一酷狗账号下载的文件才能解密。

常见问题

解密失败，提示 104003？

Cookie 里缺少 uin，到设置页「QQ 音乐 uin（QQ 号）」框填上你的 QQ 号即可。

我的文件安全吗？

所有解密都在你的浏览器本地完成，不会上传任何文件到服务器。

能离线使用吗？

可以。访问过一次后资源会自动缓存，断网也能用。

哪些格式暂不支持？

.mflach / .xmf 等网页版暂不支持，请用 CLI 版本。
